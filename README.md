# 저승 막걸리 (底承 막걸리)

> 망자(亡者)만 입장 가능한 한국 전통 심야 주막의 메뉴판 웹사이트.

## 실행 방법

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 (자동 재시작)
npm run dev

# 3. 일반 실행
npm start
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 접속.

## 요구 환경

- Node.js v20 LTS 이상
- npm v10 이상
