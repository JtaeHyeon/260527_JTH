# 저승 막걸리 — Project Memory

> Claude Code가 모든 세션 시작 시 자동으로 로드하는 프로젝트 메모리 파일.
> 본 파일은 **하네스 진입점** 역할만 하며, 실제 규칙은 3개 분리된 문서에 위임한다.

---

## 프로젝트 한 줄 소개

망자(亡者)만 입장 가능한 한국 전통 심야 주막의 메뉴판 웹사이트.
**Node.js(Express) + Bootstrap 5 + EJS** 기반.
한국 민속 귀신·요괴를 모티프로 한 다크 유머 컨셉.

---

## 하네스 문서 (Harness Documents)

본 프로젝트는 3축 하네스 구조를 따른다.
모든 작업 전 아래 문서를 반드시 참조한다.

@AGENTS.md
@SKILLS.md
@DESIGN.md

---

## 세션 진입 규칙 (Session Rules)

1. **세션 시작 시**: 위 3개 문서를 먼저 숙지한 뒤 작업을 시작한다.
2. **충돌 발생 시 우선순위**:
   - 콘텐츠 톤 결정 → AGENTS.md
   - 도구·스킬 사용 → SKILLS.md
   - 시각·디자인 → DESIGN.md
   - 위 어디에도 없는 경우 → 사용자에게 질의
3. **새 결정사항**: 일회성이 아니라 반복될 규칙이면 즉시 해당 문서에 반영한다.
4. **헥스코드 직접 사용 금지** — 컬러는 항상 DESIGN.md의 `var(--*)` 토큰 사용.

## 실행 환경 빠른 참조 (Quick Reference)

| 항목           | 값                                  |
| -------------- | ----------------------------------- |
| Node.js        | v20 LTS 이상                        |
| npm            | v10 이상                            |
| 개발 서버      | `npx nodemon server.js`             |
| 정적 자원 점검 | `npx serve ./public`                |
| 포맷 점검      | `npx prettier --check .`            |
| 기본 포트      | 3000                                |
| 엔트리         | `server.js`                         |
| 메뉴 데이터    | `data/menu.json` (단일 진실 공급원) |

---

## 사용자가 자주 쓰는 명령 (User Shortcuts)

| 사용자 명령            | 의미                                            |
| ---------------------- | ----------------------------------------------- |
| "토큰 점검해줘"        | `color-token-linter` 스킬 호출 (SKILLS.md #4)   |
| "메뉴 추가해줘 [이름]" | `menu-manager` 스킬로 menu.json에 항목 추가     |
| "카드 다시 만들어줘"   | `card-generator` 스킬로 메뉴 카드 마크업 재생성 |
| "하네스 갱신해줘"      | 최근 결정사항을 AGENTS/SKILLS/DESIGN에 반영     |

---

## 금지 사항 (Hard Rules)

- 외부 라이브러리 임의 추가 ❌
- 메뉴 정보를 EJS/HTML에 하드코딩 ❌
- CSS 헥스코드 직접 사용 ❌
- AGENTS.md 6장의 콘텐츠 금기 위반 ❌
- 사용자 승인 없는 데이터 삭제 ❌
